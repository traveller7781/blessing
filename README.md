# 母亲节祝福网页

一个温馨的互动式母亲节祝福页面。花瓣飘落、点击绽放爱心、亲手点亮五朵康乃馨。

## 使用方式

1. 将 `background-music.mp3` 放在项目根目录（可选，没有也能正常运行）
2. 浏览器打开 `index.html` 即可
3. 手机上访问：`npx serve .` 起本地服务器，手机连同一 WiFi 扫码

## 功能

- **花瓣雨**：花瓣 + 星星 + 爱心持续飘落，鼠标/手指靠近粒子会被推开
- **点击特效**：点击任意空白处绽放爱心和花瓣
- **五朵康乃馨**：点击花朵从花苞绽放为康乃馨（健康、幸福、平安、快乐、美丽），五朵全部绽放后弹出流光祝福语
- **音乐播放器**：右上角 ♪ 按钮，点击播放/暂停背景音乐，播放时音符旋转
- **装饰元素**：屏幕四周半透明花饰漂浮，卡片角花点缀
- **移动端适配**：iPhone 刘海屏安全区、响应式断点（600px / 380px）、触控优化

## 自定义

所有内容在单个 `index.html` 中，直接编辑：

| 内容 | 位置 |
|------|------|
| 卡片祝福语 | `<div class="blessing">` |
| 署名 | `<p class="signature">` |
| 全开祝福文字 | `showToast('...', true)` |
| 页面标题 | `<title>` 标签 |
| 背景音乐 | 替换 `background-music.mp3`，JS 中 `bgMusic.volume` 调音量 |
| 配色 | CSS 中的 `body background` 渐变和卡片 `background` |
| 花瓣数量 | JS 中 `MAX_BG` / `MAX_FALLING` |

## 技术

纯静态页面，零依赖。Canvas 粒子系统 + CSS 动画 + Web Audio API / HTML5 Audio。
